# Circlular-firewood-stack
I'm building a firewood stack at home using steel panels left over from our stone-cage fence. The project was given to me by my mum and I am aiming to get it done across 2 weekends with an extra weekend if required.
The design parameters my mum has provided me are to use the left-over steel cageing panels and to minimise the distance the it protrudes from our fence. I am also regularly checking in, asking questions about what she wants so I can design the solution. 
Each panel is 2400mm X 1200mm and I am using twistable cageing connectors to connect the panels. 
Smaller panels are available, which are 1100mm X 450mm
